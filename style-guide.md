# FitLife Gambia (v2) - Style Guide

## Layout

The design uses a max-width container:

- Container max-width: 1000px
- Container padding: 0 20px

## Colors

### Primary

- **Primary (Energetic Red)**: #E63946
- **Secondary (Deep Navy)**: #1D3557

### Neutral

- **Accent (Light Mint)**: #F1FAEE
- **Page Header**: #0f65df
- **Text**: #333333
- **Background**: #FFFFFF
- **White**: #FFFFFF

### Overlays

- **Hero Overlay**: rgba(29, 53, 87, 0.7)
- **Text Shadow**: rgba(0, 0, 0, 0.5)

## Typography

### Body Copy

- Font size: 16px
- Line height: 1.6

### Headings

- H1: 32px (36px in hero)
- H2: 24px
- H3: 20px

### Font

- Family: Georgia, serif
- Weights: 400 (normal), 700 (bold)

## Spacing

- Section margin-bottom: 40px
- Paragraph margin-bottom: 15px
- Container padding: 20px
- Header/Footer padding: 15px / 25px
